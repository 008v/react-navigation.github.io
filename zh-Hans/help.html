需要帮助？

如果你遇到 React Navigation 的 bug, 请 发布一个 issue 并确保填写 issue 模板。如果你认为有一个功能丢失, 请 在 Canny 上创建一个功能请求, 或者如果你正在为该功能建议一个 API 的任务, 提交一个 RFC! 如果你只是需要一些帮助, 尝试加入我们的 react-navigation 频道 Discord 或 在 StackOverflow 发布一个问题.

浏览文档和 API

请使用本站点上的文档和阅读API参考了解更多信息。

加入社区

请在 Reactiflux Discord 的 react-navigation 频道中询问有关文档和项目的问题 。

保持最新

在 Github 仓库的发布选项卡 中阅读 React Navigation 的新版本的发布说明。
